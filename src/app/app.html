What is TavernQuill?

TavernQuill is a high-performance, precision architect for SillyTavern V3 character cards. We built it to be the fastest way to refine AI character personalities, lorebooks, and sprites — with zero server interaction.

SillyTavern V3 Spec

Full support for the Spec V3 standard, including depth prompts, talkativeness settings, and enhanced metadata.

World Info & Lorebooks

Easily manage and edit world info entries. TavernQuill extracts lorebooks directly from your PNG character cards.

Privacy-First Architecture

TavernQuill is a client-side only editor. Your characters and sprites never leave your machine — no uploads.

PNG Metadata Extraction

Drag and drop any character PNG to instantly extract, edit, and re-export the embedded character data seamlessly.
